# 🎬 Netflix Clone

A simple **Netflix Clone** built using **HTML and CSS**, replicating the look and feel of the Netflix landing page. This project focuses on UI design, layout structuring, and responsiveness.

---

## 📌 Features

* 🎥 Netflix-inspired homepage UI
* 📱 Responsive design (works on mobile & desktop)
* 🎨 Clean and modern layout
* 🧭 Navigation bar with hero section
* 📺 Movie/series thumbnail sections

---

## 🛠️ Tech Stack

* HTML5
* CSS3
